How to use DataIRL:

1. Download one of template QSFs linked below.


Currently, only template surveys for CX Customer Care, CX Digital Experience, EX Candidate Experience, EX Employee Experience are available. The plan is to create more template surveys compatible with the tool based on need/interest.



2. Upload a template QSF to the brand of your choice.


Upload the QSF for the template of your choice to any Qualtrics brand. Take care to not alter the survey format in any manner or the automation will not work.


3. Generate an anonymous link from your uploaded template survey.


Navigate to the Qualtrics platform and publish the survey. From there, you can generate an anonymous link in the Distributions tab.


4. Navigate to the DataIRL tool.


Type in "datairl.app" in your Chrome browser search bar.


5. Fill out the form fields.


Enter in your email, anonymous link, the number of responses you would like generated, and select the corresponding response type from the drop-down.


6. Submit your request.


If your request has been successfully submitted, you will receive a confirmation pop-up. When your request is completed, the system will send you a "Request Complete" email to the email provided in the email block of the request.


FAQs


Q: Why does the tool require my email?

A: We require an email address, so that we can send you an email when your request has completed. We also plan to limit users to 3 response generation requests per day.


Q: I'm having trouble running this tool in my browser, is DataIRL supported on all browsers?

A: DataIRL is optimized to run on Chrome. Support for other browsers can be expected in the future.
